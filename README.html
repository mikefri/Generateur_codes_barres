GenCodeBarres v2.3 - Générateur de Planches d'Étiquettes

GenCodeBarres v2.3 est une application web statique et puissante, conçue pour générer rapidement des codes-barres et des QR Codes en série, organisés sur des planches d'étiquettes prêtes à imprimer. Idéal pour l'inventaire, la gestion d'actifs ou l'étiquetage précis de produits.

Démarrage Rapide

Le projet est entièrement *client-side* et ne nécessite pas de serveur.

  1. Clonez ou téléchargez les fichiers sources (index.html, plaquettes.js, plaquettes.css).
  2. Ouvrez index.html dans votre navigateur (ex: file:///.../index.html).
  3. Importez votre fichier Excel ou configurez vos codes, et imprimez !

Fonctionnalités Clés

Catégorie Description Détails Techniques
Importation Prise en charge de l'importation en masse via glisser-déposer d'un fichier Excel (.xlsx). Les codes sont lus depuis la colonne A.
Génération Supporte les formats de codes les plus courants. Code 128, EAN-13 (avec calcul auto. du chiffre de contrôle), QR Code.
Aperçu & Grille Configuration de grille avancée et préréglages pour un alignement parfait. Presets, contrôle précis des Marges, Hauteur d'Étiquette et Échelle du Code.
Orientation Option d'ajouter une flèche de marquage (Haut ou Bas) pour faciliter la pose. Utile pour garantir un sens de lecture ou de collage cohérent.
Exportation Exportation professionnelle garantie en PDF et impression directe. Exportation Multi-Pages vers PDF optimisé. Impression directe via CSS.

Stack Technique (Client-Side)

Ce projet est basé sur une architecture statique et utilise les librairies suivantes :

Composant Rôle Librairie Utilisée
Génération Code-barres Création de Code 128 et EAN-13 (SVG) JsBarcode
Génération QR Code Création de QR Codes (Canvas) qrcode.js
Lecture de Données Importation des codes depuis les fichiers Excel xlsx.js (SheetJS)
Exportation PDF Conversion de l'aperçu en PDF multi-pages html2canvas et jspdf
UI/UX Interface utilisateur moderne et thème sombre (Slate) HTML / CSS / JavaScript

Guide d'Utilisation Détaillé

Étape 1 : Préparation des Données

Vous avez deux options pour fournir vos codes :

  • Option 1: Import Excel (Recommandé pour le volume)
    1. Créez un fichier Excel (.xlsx).
    2. Listez tous les codes (numériques ou alphanumériques) dans la Colonne A du fichier.
    3. Glissez-déposez le fichier sur la zone **"Import .xlsx"** de l'application. Le statut confirmera le nombre de codes chargés.
  • Option 2: Saisie Manuelle

    Utilisez la section de saisie manuelle pour quelques codes.

Étape 2 : Configuration de la Planche

  1. Type de Code : Sélectionnez le format souhaité (Code 128, EAN-13 ou QR Code).
  2. Dimensions :
    • Choisissez un Modèle prédéfini (Preset) pour une configuration standard.
    • OU ajustez manuellement les dimensions (Marges, Colonnes, Lignes, Hauteur d'Étiquette) en millimètres (mm) pour qu'elles correspondent exactement à vos étiquettes vierges.
  3. Visualisation : L'aperçu est mis à jour en temps réel. Utilisez le curseur de Zoom pour vérifier votre alignement.

Étape 3 : Impression et Exportation

Une fois satisfait de l'aperçu :

Télécharger le PDF

Lance la conversion de toutes les pages générées en un seul fichier PDF. C'est le format recommandé pour une impression de haute qualité ou pour l'envoi à un prestataire.

Imprimer toutes les Planches

Lance directement la boîte de dialogue d'impression. Le CSS intégré garantit que seule la planche (sans l'interface) sera imprimée.
